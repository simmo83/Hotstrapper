# HotStrapper

**For rapid web development using Gulp + Sass + BrowserSync**

## Getting started
### Install NodeJS
Install from [https://nodejs.org/en/download/](https://nodejs.org/en/download/)

### Install GulpJS

Install globally

`npm install -g gulp`

### Install ImageOptim

Install from [https://imageoptim.com/mac](https://imageoptim.com/mac)
